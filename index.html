Gratis website på GitHub Pages

(Fungerer til dine egne håndkodede websites - ikke til Wordpress etc.)

  1. Lav et nyt repository (med GitHub Desktop)
  2. Byg dit site (lokalt) og gem det i repository-mappen
  3. Commit (med GitHub Desktop)
  4. Log in på GitHub i en browser
  5. Find dit repository og klik på Settings
  6. Gør dit repository "public" nederst under Danger Zone
  7. Klik på Pages i menuen til venstre
  8. Vælg "main" og "root" under Branch og klik Save
  9. Vent 2min...
  10. Dit site er nu online her: https://dit-github-navn.github.io/dit-repository-navn
